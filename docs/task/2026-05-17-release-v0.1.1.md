# Release v0.1.1 — 누적 변경분을 새 portable .exe로 배포

> 날짜: 2026-05-17 · 커밋: TBD · 릴리즈: TBD

## Context

v0.1.0 release(2026-05-16) 이후 다음 변경들이 main에 쌓였는데, 첨부된 .exe는 여전히 v0.1.0 빌드라 다운로더가 옛 동작을 보게 됨:

- Launcher Linear/Geist 재디자인 + 다크/라이트 토글
- 한국어 카피 일관화
- "강도" → "우선 순위", 슬라이더 라벨
- 말풍선 항상 표시 + 맑은 고딕 통일
- 컨트롤 툴팁
- 최소화 → 재설정
- **60분 타이머 정확도 버그** 2종 수정 (closure 캡처 + round/ceil)
- npm start 자동 재빌드

사용자가 "git에 배포는 완료 했나요?"라고 묻고 v0.1.1을 만들도록 지시.

## What changed

- `package.json` — `version`을 `0.1.0` → `0.1.1`로 올림
- `release-notes-v0.1.1.md` — 사용자용 변경 노트 (gitignore되어 commit 안 됨)
- 빌드 산출물: `release/0.1.1/Focus Companion 0.1.1.exe` (portable, Electron 기본 아이콘)
- GitHub Release `v0.1.1` 신규 생성 — 위 .exe 첨부

## Files touched

## How to verify

1. https://github.com/ccomajeon/focus-companion/releases/tag/v0.1.1 에 접속
2. Assets에 `Focus Companion 0.1.1.exe` 첨부되어 있는지 확인 (SHA-256 일치)
3. .exe 다운로드 → 더블클릭 → Launcher가 새 디자인(다크 슬레이트 + 바이올렛)으로 뜨는지 확인
4. 60m 선택 → Begin Focus → 위젯 첫 frame이 정확히 `60:00` 인지 확인
5. 위젯 호버 시 ⏸/↻/✕ 툴팁 한글 표시 확인
